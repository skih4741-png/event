# Changelog

## v0.3 — 2026·05·13 (Phase 1 · 완성판)

**핵심: 산출물 자동 생성 + 디자인 테마 + URL 자동 연결 + 인터랙티브 매뉴얼**

### ⭐ NEW · 산출물 생성 페이지
기획안 한 건이면 9종 산출물이 자동 생성됩니다.

**마케팅 카피 (5채널 × 5톤 × 변형 2~3안 · 풀 100개+)**
- 앱 푸시 (헤드 25자/본문 50자 자동 검증)
- SMS / LMS (단문 90자 / 장문 2,000자)
- 카카오 알림톡
- 이메일 (제목·프리헤더·본문·CTA 분리)
- 인앱·웹 배너 카피
- 톤 5종: 혜택강조 / 한정성 / 친근 / 격식 / **🔥 트렌디** (NEW)
- 시드 기반 셔플 — `↻ 새로 생성` 누를 때마다 다른 조합
- 각 카피 옆 ⌘ 복사 버튼
- 카피 카드에 적용된 URL 표시

**비주얼 미리보기 (10종 mockup, 실제 HTML/CSS 렌더)**
- iOS 푸시 / Android 푸시
- 인앱 배너 3종 (상단·중앙·하단)
- 웹 배너 PC·모바일
- 카카오톡 채널 메시지
- 이메일 HTML 카드 (실제 발송 코드 복사 가능)
- 랜딩 페이지 와이어프레임
- PNG 다운로드 (html2canvas) / HTML 코드 복사 / 인쇄

**운영 문서 (4종, A4 인쇄 가능)**
- 1페이지 기획서 (회의 배포용)
- D-7 ~ D+2 운영 캘린더 (체크리스트 자동 배치)
- 결과 보고서 (KPI · ROAS 자동 채움 + 회고 영역)
- 회고 노트 (Keep · Problem · Try)

- 기획안별로 저장

### 🔗 NEW · 랜딩 URL + UTM 자동 추가
- 기본 URL 입력 → 모든 CTA 자동 연결
- 채널별 다른 URL 지정 (펼치기/접기)
- UTM 파라미터 자동 추가
  - `utm_source` · 채널명 (push/email/sms/kakao/banner)
  - `utm_medium` · 매체 (notification/email/sms/social/display)
  - `utm_campaign` · 기획안 ID
- UTM on/off 토글
- 딥링크 자동 감지 (`tripplus://...` 은 UTM 미적용)
- https:// 자동 prepend
- 채널별 URL 미리보기 표시
- 이메일·카톡·배너 CTA가 실제 `<a href>` 로 렌더링 (HTML 복사 시 그대로)

### 🏢 NEW · 서비스/회사 정보 인라인 편집
- 산출물 페이지에서 직접 서비스명·회사명·발신자명 변경
- 설정 페이지와 양방향 자동 동기화
- 모든 산출물에 즉시 반영

### ⇅ NEW · 섹션 접기
- 서비스 정보 / 디자인 테마 / 랜딩 URL 각각 개별 접기 토글
- `⇅ 설정 접기` 버튼 — 한 번에 전체 접기/펼치기
- 접힘 상태 LocalStorage 자동 저장
- 페이지 재진입 시 그대로 복원

### 💾 NEW · 자동 저장 상태 인디케이터
- 산출물 페이지 우측 상단 배지
- 평소: `💾 자동 저장됨` (초록)
- 입력 중: `💾 저장 중...` (노랑, 점멸)
- 완료: `✓ 방금 저장됨` (초록, 3초)
- 별도 저장 버튼 없음

### ✨ KPI 정의 단계 개선
- 6개 빠른 선택 칩 (DAU / 매출 / 신규 가입 / 재방문율 / 전환율 / 휴면 재활성)
- 각 칩에 부연 설명 표시
- 클릭 한 번에 KPI 항목·목표·단위 자동 입력
- 시각적 피드백 (선택 시 강조)
- "🤔 어떻게 정해야 할지 모르겠다면?" 접고 펴기 도움말
- 단계별 예시 + 추천 가이드

### 📊 NEW · 컬럼 매핑 UI 강화
- 신뢰도 < 70 또는 자동 매핑 실패 시 모달 자동 표시
- 컬럼별 진단 정보 (타입·고유값·결측%·샘플)
- 미리보기 표 (10행)
- 매핑 프로필 저장 / 불러오기
- **재매핑 가능** · 차트 본 뒤 ✎ 매핑 수정으로 컬럼 변경, 재업로드 불필요
- 메모리 + LocalStorage 양쪽 보관 (5,000행 이하 자동 저장)

### 💬 NEW · 모달 폼
